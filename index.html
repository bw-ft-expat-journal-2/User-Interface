Mission Statement

Our mission at Expat Journal is bringing the world to adventurers and discoverers fingertips with originality, imagination and creativity...

Are you on vacation at the beach, or maybe taking a challenging climb in the mountains? Share your most iconic and memoriable times through imagery with Expat Journal. Our team is dedicated to bringing you the best sharing experience the internet has to offer!

Features

User Management

Registering with Expat Journal is simple and through the creation of your own account you can post and store all your trips photos and stories!

Post and Story Sharing

Once you have created your own account you can start to share your travel adventures with the community! Let others live vicariously through your travels by posting images and stories!

View Others Posts and Stories

Get inspired by the posts of others! Checkout other user stories and images as you broaden your horizon and learn about new places and people!

Build Framework

React

React makes it painless to create interactive UIs. Design simple views for each state in your application.

Redux

Redux helps you write applications that behave consistently, run in different environments, and are easy to test.

Node JS

As an asynchronous event-driven JavaScript runtime, Node.js is designed to build scalable network applications.

Axios

Axios allows you to make XMLHttpRequests from the browser, http requests from node.js, supports the Promise API and intercepts request and responses.

Cypress

Cypress is the new standard in front-end testing that every developer and QA engineer needs.

Formik

Formik is the world's most popular open source form library for React and React Native.

Jest

Jest is a delightful JavaScript Testing Framework with a focus on simplicity. Jest works right out of the box in many environments.

Knex

Knex.js is a "batteries included" SQL query builder for Postgres, MSSQL, MySQL, MariaDB, SQLite3, Oracle, and Amazon Redshift designed to be flexible, portable, and fun to use.

Yup

Yup is a simple object schema validator I came across recently. It is inspired by Joi.
